    หน้า detail ตรวจรายชื่อ guest แบบไม่แก้จริงได้

ฟีเจอร์ · OT / ค่าล่วงเวลา

OT Module — มี state machine + ลายเซ็น

  • 2 ประเภท
    HOLIDAY (วันหยุด = 1 วัน) · OVERTIME (เกิน 8 ชม. = นับชั่วโมง)
  • workflow อนุมัติ
    DRAFT → SUBMITTED → APPROVED / REJECTED · reject แล้วแก้ส่งใหม่ได้
  • ลายเซ็นดิจิทัล
    snapshot ลายเซ็นผู้ขอ + ผู้อนุมัติ เก็บติดเรคคอร์ด (แก้ทีหลังไม่กระทบของเก่า)
  • auto จาก booking
    OT ที่เกิดจากงานผูก bookingId ได้ · ผูกรายเดือน YYYY-MM
